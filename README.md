# EvalXpert: Your Powerful and Intelligent Calculation Partner EvalXpert is a hybrid calculator application built with Node.js, offering both a Command Line Interface (CLI) for quick calculations and a beautiful Web User Interface (Web UI) for intuitive, visual interaction.  ## Features - **CLI Utility:** Quickly evaluate mathematical expressions directly from your terminal. - **Web UI:** A modern, responsive web interface inspired by 'AquaFlow' design principles for an elegant user experience. - **Robust Calculation Engine:** Powered by `mathjs` for accurate and secure evaluation of complex expressions. - **Basic Operations:** Supports addition, subtraction, multiplication, division, and parentheses. - **History (Web UI):** Keep track of your recent calculations in the web interface.  ## Getting Started  ### Prerequisites Make sure you have Node.js installed on your system. You can download it from [nodejs.org](https://nodejs.org/).  ### Installation 1. **Clone the repository:** ```bash git clone https://github.com/your-username/evalxpert.git cd evalxpert ``` 2. **Install dependencies:** ```bash npm install ```  ## Usage  ### 1. Command Line Interface (CLI) To use EvalXpert as a CLI tool, run the `cli.js` script with your mathematical expression as an argument.  ```bash node cli.js "10 * (2 + 3) / 2" ```  **Examples:** ```bash node cli.js "1 + 1" # Output: 2 node cli.js "(5 + 3) * 2 - 10 / 5" # Output: 14 node cli.js "sqrt(16) + pi" # Output: 7.141592653589793 ```  ### 2. Web Application To run the EvalXpert web application: 1. **Start the server:** ```bash npm start ``` 2. **Open your browser:** Navigate to `http://localhost:3000` (or the port specified in your `.env` file).  You will see a beautiful calculator interface where you can input expressions using buttons or your keyboard, and view results and history.  ## Project Structure ``` /evalxpert/ ├── README.md                               # Project overview and usage ├── package.json                            # Project metadata and dependencies ├── .env.example                            # Environment variable template ├── .gitignore                              # Files/directories to ignore by Git ├── server.js                               # Express server entry point (Web UI, API) ├── cli.js                                  # CLI tool entry point ├── src/                                    # Application's core logic │   ├── calculator/                         # Math expression evaluation modules │   │   ├── parser.js                       # (Delegates to mathjs) │   │   ├── evaluator.js                    # (Delegates to mathjs) │   │   └── index.js                        # Public interface for calculator logic │   └── api/                                # Web API endpoint definitions │       └── calculate.js                    # /api/calculate endpoint logic └── public/                                 # Web UI static files │   ├── index.html                          # Main page for the web calculator │   ├── css/                                # Stylesheets │   │   └── main.css                        # Compiled CSS for the web UI │   └── js/                                 # Frontend JavaScript │       ├── app.js                          # Main frontend logic (UI events, API calls) │       └── ui.js                           # UI rendering/update helper functions ```  ## Contributing Contributions are welcome!
